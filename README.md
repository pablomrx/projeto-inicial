# Java: Criando sua Primeira Aplicação ☕

Repositório com exercícios e desafios desenvolvidos durante o curso **"Java: criando sua primeira aplicação"** da Alura.

## 📚 Conteúdos estudados

- Configuração do ambiente Java (JDK 26 e IntelliJ IDEA)
- Variáveis e tipos primitivos
- Operadores aritméticos e lógicos
- Estruturas condicionais (`if` e `switch`)
- Estruturas de repetição (`for` e `while`)
- Entrada de dados com `Scanner`
- Formatação de saída com `printf`
- Casting de tipos
- Lógica de programação

## 🛠️ Tecnologias utilizadas

- Java 26
- IntelliJ IDEA
- Git
- GitHub

## 📂 Exercícios desenvolvidos

- Conversor de temperatura
- Sistema bancário simples
- Exercícios de lógica
- Estruturas de repetição

## 🎯 Objetivo

Praticar os fundamentos da linguagem Java e desenvolver a base da lógica de programação.
